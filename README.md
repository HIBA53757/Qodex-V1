# Qodex-V1
## Description
Qodex_v1 est une application web de gestion de quiz destinée aux enseignants.  
Elle permet de créer des quiz, gérer les questions et consulter les statistiques.

## Technologies utilisées
- PHP (POO)
- MySQL
- HTML / CSS
- JavaScript (vanilla)

## Fonctionnalités
- Authentification enseignant
- Création et gestion des quiz
- Ajout de questions
- Gestion des catégories
- Consultation des résultats et statistiques
